# Techcean

A modern multi-page website built with React and Vite.

## Tech Stack

- **React** 18.2.0
- **React Router DOM** 6.22.0
- **Vite** 5.1.4
- **CSS Modules** for styling

## Pages

- **Home** - Landing page with hero, services preview, process overview
- **Services** - Services offered
- **Work** - Portfolio/work showcase
- **Process** - Business process explanation
- **Contact** - Contact information

## Live Demo

View the website: https://techcean-brown.vercel.app/

```bash
# Install dependencies
